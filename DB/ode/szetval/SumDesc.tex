Egy \de{et} szétválasztható változójúnak nevezünk, ha
\Mat{
   g(y)y'=f(x)
}
alakú, vagy ilyenre hozható. Vagyis az $x$ és $y$ változók elkülöníthetőek 
(szétválasztható, szeparálható)\newline
A fenti alak 'megoldása':
\Mat{
    \mint{g(y)}{y}=\mint{f(x)}{x}
}
A következő speciális esetekkel gyakran találkozunk:
\Mat{
   y'=f(x) \hspace{1cm} y=\mint{f(x)}{x} \us
   y'=g(y) \hspace{1cm} x=\mint{\frac{1}{g(y)}}{y} \us
   y'=f(x)g(y) \hspace{1cm} \int f(x)dx=\mint{\frac{1}{g(y)}}{y}
}
