Jelölje $s(t)$ a tartálybeli só mennyiségét $t$-edik időpillanatban. Legyen $\Delta t$
egy "elegendően" kicsiny időtartam. Ekkor
\Mat{
   s(t+\Delta t)=s(t)-\frac{s(t)5\Delta t}{100}
}
A $\Delta t \to 0$ határátmenetet véve kapjuk a
\Mat{
   s'=-\frac{5}{100}s
}
\de{et,} melynek általános megoldása
\Mat{
   s(t)=Ce^{-\frac{5}{100}t}
}
melyből,
\Mat{
s(0)=0.5=Ce^0=C
}
adódik, ahonnan
\Mat{
s(60)=\frac{0.5}{e^3}\approx \ 0.025 \text{kg}.
}
